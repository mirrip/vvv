# V Messenger

A new messenger app applies for Android and iOS.

## Download

- [V-messenger APK](https://raw.githubusercontent.com/mirrip/vvv/main/frontend/apk_output/v-messenger.apk)

## Features

- Real-time messaging with Socket.IO
- Chats, groups, and channels
- Media: text, photos, video, files, word and video messages
- Reactions
- Profiles and avatars
- Real-time statuses

## Note

This APK is built with a minimal android app template and includes the JavaScript bundle. To run android, you need the backend server running on your local machine.

See below for starting the backend.

## Backend (Server)

```bash
# backend
cd backend
npm install
npm start
```

## Frontend (Expo)

```bash
# frontend
cd frontend
npm install
npm start
```

The app would be operational. The apps for Android and iOS have to be built using Easo for the final distribution.
